# 🦅 Bali Enterprises | Global Holding & Investment Gateway

Bali Enterprises is a premium, consulting-grade ecosystem facilitating international investment and strategic market entry into Indonesia. This platform serves as the central intelligence node for the Bali Enterprises Group (including PT Indonesian Visas Agency / MYVISA and PT Bali Experience Group).

---

## 🛡️ Production Status: **HARDENED**
The platform has undergone a comprehensive production-grade hardening phase (Q2 2026), achieving absolute synchronization between its visual identity, legal requirements, and SEO architecture.

### **Key Ecosystem Assets**
- **Strategic Entry Blueprint:** The definitive roadmap for foreign founders.
- **BPOM Certification Guide:** High-fidelity compliance roadmap for regulated industries.
- **Global Data Hub:** 85+ pre-rendered static routes covering every major economic hub in Indonesia.

---

## 🛠️ Technology Stack
- **Engine:** Next.js 15.3 (App Router)
- **Typology:** Outfit (Branding) & Inter (UI)
- **Styling:** Tailwind CSS 4.0 + Custom Glassmorphism System
- **Structure:** Schema.org JSON-LD (Synchronized Master Node)
- **Icons:** Lucide React
- **Performance:** Sharp (Image Processing) & Framer Motion (Animations)

---

## 🚀 Development & Build

### **Prerequisites**
- Bun (Preferred) or Node.js 20+

### **Commands**
```bash
# Install dependencies
bun install

# Local development
bun run dev

# Production hardening build
bun run build

# Start production standalone server
bun start
```

## 🗺️ Sitemap Architecture
The ecosystem is structured into 7 distinct clusters:
1. **Core Guides:** Strategic entry and business starting guides.
2. **National Economic Hubs:** Setup guides for major cities (Jakarta, Surabaya, etc.).
3. **Regional Growth Centers:** Specialized zones (Batam, Labuan Bajo, etc.).
4. **Entrepreneurship & Culture:** Localized business environment data.
5. **Investment Strategy:** Macroeconomic benchmarks for Indonesia.
6. **Corporate Services:** The internal Bali Enterprises service stack.
7. **Jurisdictional Comparisons:** Benchmark data against Thailand, Singapore, and Dubai.
